
# 🚀 Study Apex
Sistema de gerenciamento de estudos focado em alta performance.

## 📊 Funcionalidades
- **Dashboard Interativo**: Acompanhamento diário, semanal, mensal e anual.
- **Cronômetro de Foco**: Timer integrado para sessões de estudo.
- **Registro Manual**: Adicione horas de estudo retroativas.

## 🛠️ Tecnologias
- HTML5 / CSS3 / JavaScript
